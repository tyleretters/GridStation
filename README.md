## Local development

This is a static site. No build step.

1. Clone the repository.
2. From the repo root, run `npx serve`.
3. Open the URL it prints (usually [http://localhost:3000](http://localhost:3000)).

jQuery and Bootstrap are vendored under `assets/`. The site has no external runtime dependencies.
